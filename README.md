# SAT Home & Lifestyle | Product Landing Page

A modern, responsive product landing page for SAT Home & Lifestyle, showcasing architectural, construction, and bespoke furniture services. This project was built as part of the freeCodeCamp Responsive Web Design curriculum.

## Features

- **Hero Section:** Engaging introduction with a call-to-action for free quotes.
- **Services:** Highlights expertise in architecture, construction, and custom furniture.
- **Design Process:** Outlines the client journey from consultation to execution.
- **Portfolio:** Visual gallery of completed projects (images in the `images/` folder).
- **Pricing:** Clear project tiers for different client needs.
- **Footer:** Company info, quick links, services, contact details, and an inquiry form.

## Demo
Open `index.html` in your browser to view the landing page.

## Folder Structure
```
Product Landing Page/
├── index.html
├── style.css
└── images/
```

## How to Use
1. Clone or download this repository.
2. Open the `Product Landing Page` folder.
3. Open `index.html` in your web browser.
4. All styles are in `style.css`. Images are in the `images/` folder.
